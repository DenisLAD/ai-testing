# Руководство по UI для AI Testing Platform

## Новые возможности

### 1. Запуск тестовой сессии

**Интерфейс:**
- Поле URL для указания тестируемого сайта
- Выбор цели тестирования (EXPLORATORY, REGRESSION, SECURITY, ACCESSIBILITY)
- Текстовое поле для описания тестового сценария
- Кнопка "Запустить тестирование"

**Пример описания сценария:**
```
Протестируйте функционал входа с неверными учетными данными.
Попробуйте ввести неверный логин и пароль, проверьте появление сообщения об ошибке.
```

### 2. Остановка сессии

**Кнопка "Остановить"** становится активной во время тестирования.

- Останавливает выполнение тестов
- Устанавливает статус сессии в STOPPED
- Сохраняет выполненные действия

**API:** `POST /test/stopSession/{id}`

### 3. Просмотр всех сессий

**Кнопка "Все сессии"** открывает модальное окно со списком всех сессий.

**Таблица сессий содержит:**
- ID сессии
- URL тестирования
- Описание сценария
- Цель тестирования
- Статус (RUNNING, COMPLETED, STOPPED, FAILED)
- Количество выполненных действий
- Длительность сессии
- Кнопки действий (просмотр, генерация теста)

**Функции:**
- Клик по строке выбирает сессию для работы
- Кнопка "Обновить" обновляет список сессий
- Кнопка генерации теста доступна только для завершенных сессий

### 4. Генерация теста

**Кнопка "Сгенерировать тест"** доступна для завершенных сессий.

**Процесс:**
1. Нажать кнопку "Сгенерировать тест"
2. Дождаться генерации (отображается спиннер)
3. Просмотреть информацию о тесте:
   - Имя класса
   - Пакет
   - Количество методов
4. Просмотреть код теста в формате Java
5. Скачать тест кнопкой "Скачать"

**API:** `POST /test/{id}/generate-test`

**Пример сгенерированного теста:**
```java
@TestMethodOrder(MethodOrderer.OrderAnnotation.class)
public class LoginInvalidCredentialsTest {
    
    private static WebDriver driver;
    private static WebDriverWait wait;
    
    @Order(1)
    @Test
    @DisplayName("Навигация на страницу логина")
    void navigate_to_01_page() {
        driver.get("https://the-internet.herokuapp.com/login");
    }
    
    @Order(2)
    @Test
    @DisplayName("Ввод username")
    void type_02_username() {
        WebElement element = driver.findElement(By.id("username"));
        element.sendKeys("invalid_user");
    }
}
```

### 5. Скачивание теста

**Кнопка "Скачать тест"** загружает файл `.java` с сгенерированным тестом.

**API:** `GET /test/{id}/test/download`

### 6. Лог действий

**Автоматическое обновление:**
- Каждое действие агента отображается в логе
- Цветовая индикация:
  - 🟢 Зеленый - успешные действия
  - 🔵 Синий - информационные сообщения
  - 🔴 Красный - ошибки
- Автопрокрутка включена по умолчанию

### 7. Скриншот страницы

**Автоматическое обновление:**
- Скриншот обновляется после каждого действия
- Отображается время последнего обновления
- Показывается разрешение экрана

### 8. Индикатор состояния

**Статусы:**
- 🟡 IDLE - готов к работе
- 🔵 TESTING - выполняется тестирование
- 🟢 COMPLETED - тестирование завершено успешно
- 🔴 STOPPED - остановлено пользователем
- 🟠 FAILED - произошла ошибка

## API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/test/startSession` | Запуск новой сессии |
| POST | `/test/stopSession/{id}` | Остановка сессии |
| GET | `/test/sessions` | Список всех сессий |
| GET | `/test/session/{id}` | Детали сессии |
| POST | `/test/{id}/generate-test` | Генерация теста |
| GET | `/test/{id}/test/download` | Скачать тест |

## Советы по использованию

1. **Перед запуском:**
   - Убедитесь что AI сервер запущен (localhost:1234)
   - Проверьте корректность URL
   - Детально опишите тестовый сценарий

2. **Во время тестирования:**
   - Следите за логом действий
   - При необходимости остановите тестирование
   - Скриншоты помогают понять что происходит

3. **После завершения:**
   - Просмотрите список действий
   - Сгенерируйте тест
   - Проверьте сгенерированный код
   - При необходимости отредактируйте тест

4. **Управление сессиями:**
   - Используйте модальное окно для просмотра истории
   - Выбирайте завершенные сессии для генерации тестов
   - Очищайте старые сессии периодически

## Устранение проблем

### Тестирование не запускается
- Проверьте что AI сервер доступен
- Проверьте логи приложения
- Убедитесь что URL корректный

### Генерация теста не работает
- Убедитесь что сессия завершена (статус COMPLETED)
- Проверьте что сессия содержит действия
- Проверьте логи генерации

### Скриншоты не обновляются
- Проверьте настройки браузера
- Убедитесь что достаточно памяти
- Обновите страницу (F5)
